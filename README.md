# 🎬 FilmZone - Online Movie Ticket Booking System

FilmZone is a full-stack MERN web application designed to provide users with a seamless online movie ticket booking experience. Users can browse movies, view show timings, select seats, make secure bookings, and manage their profiles—all through an intuitive digital platform.

---

## 🚀 Features

### 👤 User Features
- User registration and login authentication
- Browse latest movies and show details
- Search movies by title or category
- View available theaters and show timings
- Interactive seat selection system
- Online ticket booking
- Booking confirmation and ticket generation
- User profile management
- Booking history tracking

### 🛠️ Admin Features
- Admin dashboard
- Add, update, or delete movies
- Manage theaters and show schedules
- View user bookings
- Monitor system operations
